чтоб открыть весь воркспейс сразу - нажать synth.code-workspace

перед запуском зайти в /venv/pyvenv.cfg и вписать СВОЙ путь к интерпретатору питона (файл можно открыть прямо в vscode)
в папке с этим файлом открыть терминал и выполнить git update-index --assume-unchanged .\pyvenv.cfg чтобы локальные настройки не пушились и не сбивали настройки другим людям

для запуска виртуального окружения в терминале выполнить .\venv\Scripts\activate

после выполнения всех этих действий запустить gui.py 

оно подтянет port.py и все должно заработать
